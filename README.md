# VC Fund Game

VCファンドのパートナーとなり、10年ファンドを運用するボードゲームアプリケーション。スタートアップへの分散投資を通じて、**パワーロウ（べき乗則）** に基づくVC投資の本質を体感的に学べます。

## 特徴

- 2〜6人でプレイ（1台の端末を囲むホットシート型）
- 1ゲーム 20〜30分
- サーバー不要、ブラウザだけで完結する静的SPA
- VC投資の意思決定を疑似体験できる教育ゲーム

## ゲーム概要

- 各プレイヤーは100億円のファンドを運用するジェネラルパートナー（GP）
- 10ラウンド（10年間）でスタートアップに投資し、Exit（IPO/M&A）を通じてリターンを最大化
- ファンドDPI（分配総額 / 投資済み総額）で勝敗を競う

### 学べるVC投資の原則

| 原則 | ゲーム内の再現 |
|------|-------------|
| パワーロウ | 大半の案件は失敗。少数のホームランがファンドリターンを決める |
| 打席数の重要性 | 投資件数が少ないとホームランに出会う確率が下がる |
| フォローオンの戦略性 | 勝ち馬に追加投資して持分を維持する判断 |
| 損切りの重要性 | 苦戦案件に固執せず早めにライトオフする判断 |
| バリュエーション規律 | 高値掴みするとリターン倍率が下がる |
| セクター分散 | 特定セクターに偏ると市場イベントで大ダメージ |

## 技術スタック

- **ビルドツール**: Vite
- **フレームワーク**: React
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **状態管理**: useReducer + Context
- **グラフ**: Recharts

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

ビルド後は `dist/` フォルダの静的ファイルのみで動作します。GitHub Pages等での配信も可能です。
